A–J Guideline Reference ×
A – Bill of Lading (BOL) Disputes
A1: Missing BOL – No debtor name
Advise customer: document on file is the only document provided at pickup. No debtor name shown → standard response applied.
A2: BOL with FedEx stamp but wrong debtor
If customer is not shipper, advise LOA needed from new debtor. If shipper, advise standard denial.
A3: BOL Review required – under 30 days
Check A–J guidelines before review. Send email to origin center if BOL missing. Retrieve via Image Viewer or EDM.
B – Delivery Receipt (DR) / POD
B1: Spotted or Driver Spotted tag
Email destination service center for missing POD. If over 60 days, remove disputed charges.
B2: Delivery fee dispute (Liftgate, Inside Delivery, Sort & Segregate)
Request POD from service center. If invoice >45 days and no POD, remove charges.
C – Cargo Care / Return Delivery (CC/RD)
C1: Storage or redelivery fee dispute
Check if supporting document on file. If not, attempt retrieval. If >90 days without doc, write off.
D – Inspection & Reweigh
D1: Inspection keyword present (CAPL, CCD, FUVI, ADIM, REWE)
Move to Freight Research – Inspection Review. For REWE keyword, follow missing reweigh certificate process.
D2: Reweigh certificate missing
If PRO contains REWE keyword, move to Reweigh Review (non-reweigh agent). If already in review, follow missing certificate process.
E – LOA / LOD (Letter of Authority/Denial)
E1: LOA requested but not imaged
If rebilled via PAUD and >2 weeks, email RACF owner for actual LOA. If previous case, retrieve via ODC or TL.
E2: LOA forwarded to imaging@fedex.com
Take screenshot, attach to response, and wait for imaging. If >1 month, email client to coordinate.
F – General Age Rules
F1: PRO age <30 days
Follow BOL review / DR retrieval immediately. Use Image Viewer.
F2: PRO 30-90 days
Send email only to origin center managers (do not copy customer). If invoice >90 days, move to Invoice Reprint.
F3: PRO >90 days
Document likely not retrievable. Advise customer to contact shipper; if shipper, move to Invoice Reprint.
Sending Emails – 3‑Step Process ×
Step 1: Identify the correct recipient

Email ONLY the Origin Center Manager or Supervisor – do NOT copy the customer. Use the service center code from the PRO (e.g., ATL, DFW).

Step 2: Write a clear request

Include PRO number, missing document type (BOL/DR/CC/RD), and request they search their local records. Attach any customer‑provided reference.

Step 3: Follow up & document

If no reply within 3 business days, escalate to District Operations Manager. Once document is retrieved, send immediately to imaging@fedex.com with “IMAGING REQUEST – PRO XXXXX” in subject.
