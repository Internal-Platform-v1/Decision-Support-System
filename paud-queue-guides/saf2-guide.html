Entering Correction Comments and Correction Codes×

Enter Correction Comment: VS-SAF2-AUDT + Specific Correction Comment below.

A. Updates Entered and All is Now Billing Correctly

  • Click Autorater – Shift + F6
  • Enter Correction Comment
  • Enter Keyword CORR
  • Enter Description ACCR or ECD
  • Click Accept All – F6

B. Billing Entered Incorrectly from the Beginning and Billing was Updated

  • Click Autorater – Shift + F6
  • Enter Correction Comment
  • Enter Keyword CORR
  • Enter Description CUSI
  • Click Accept All – F6

C. Accessorial Charges are REMOVED per BOL or DR

  • Enter Correction Comment
  • Enter Keyword CORR
  • Enter Description NACC
  • Click Accept All – F6

D. After Review of BOL, if there are NO CHANGES

  • Enter Correction Comment: VS-SAF2-AUDT – NOAC RELEASING PRO FROM SAF2 QUEUE
  • Enter Keyword CORR
  • Enter Description VRFY
  • Click Accept All – F6

E. Terms are Updated per BOL

  • Click Autorater – Shift + F6
  • Add NOAR
  • Enter Correction Comment
  • Enter Keyword CORR
  • Enter Description ETMS (if Biller Error)
  • Click Accept All – F6
Always ensure the correction comment begins with VS-SAF2-AUDT followed by the specific comment for the action taken.
