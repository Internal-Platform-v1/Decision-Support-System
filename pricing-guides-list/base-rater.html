Base Rater Workflow

Follow the steps below to confirm account setup, identify shipment information, enter ZIP codes, apply class logic, itemize weight, select LTL rate family, and choose the correct tariff year.

How to Use the CZAR Rater Website
Rating Inputs
1

Open the PRO in FBI

Open the PRO in FBI and identify the shipment information needed for the rater.

2

Complete City Lookup

Enter the required ZIP codes:

  • Origin — Shipper's ZIP Code
  • Destination — Consignee's ZIP Code
3

Enter the correct class

If Pricing involves FAK Rates, enter the class that applies to the customer's class group.

Example
If the agreement states that Class 100 applies to shipments with Class 055, 060, 077, and 090, enter Class 100 instead of the actual class listed on the BOL.
4

Enter the weight

Enter the weight per line item.

Note
If the shipment contains multiple line items, click "Add New Row" and itemize the weight and class per line item.
5

Select the rate family

Select LTL as the Rate Family.

6

Select the available tariff

Select the rater used in the customer's agreement, including the applicable year.

Examples
FXF 1000 Eff 11/01/2010
or
CZARLITE Eff 02/01/2010
Important
If the correct base rater is not available, send an email to the Training Team / SMEs and include all shipment details so they can verify the correct CWT rate.
Completed
Use the correct shipment ZIPs, class, itemized weight, rate family, and tariff year to verify the base rate accurately.
